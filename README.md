This website is a personal blog and video game recommendation platform built entirely with HTML, CSS, and JavaScript. It allows the user to create, edit, and delete blog posts, each with a title, content, and timestamp. The Video Game Recommendations (VGR) section features a preloaded list of games—including No Man’s Sky, Terraria, Trailmakers, Bloxd, Hollow Knight, Hollow Knight: Silksong, Grimhook, Mech Arena, Geometry Dash, and Touhou: New World—with full details such as description, price, genre, platform, and rating.

Admins can log in with a password to access editing privileges, allowing them to add or modify games and blog posts directly in the browser, while regular users can view and search content without editing rights. All data is stored in the browser’s localStorage for persistence across sessions. The site features a consistent design with a colorful background, rounded content containers, and centered search bars, providing a clean and user-friendly interface.
